===============================================================================
                       PROYECTO: FOCOÚNICO (Versión 1.0)
                 Planificación, Estructura y Mapa de Tareas
===============================================================================

[DESCRIPCIÓN DEL PROYECTO]
FocoÚnico es una aplicación web minimalista diseñada como el antídoto definitivo
contra la procrastinación y el burnout estudiantil. A diferencia de las listas
de tareas tradicionales que abruman y generan ansiedad, FocoÚnico protege el
foco del usuario ocultando la lista global y forzando la atención en un único
sub-paso de corta duración, seguido de un descanso obligatorio.

-------------------------------------------------------------------------------
1. FILOSOFÍA DEL MVP (Producto Mínimo Viable)
-------------------------------------------------------------------------------
* Sin registros ni inicios de sesión (evita la parálisis por elección del stack).
* Sin base de datos persistente en la primera versión (todo ocurre en memoria/local).
* Interfaz ultra-limpia (cero distracciones visuales).
* Regla estricta: Una sola cosa en pantalla a la vez.

-------------------------------------------------------------------------------
2. ARQUITECTURA SIMPLIFICADA DE ARCHIVOS
-------------------------------------------------------------------------------
Para mantener el proyecto limpio y fácil de abordar, utilizaremos:
├── index.html      # Estructura de las pantallas (Volcado y Foco)
├── styles.css      # Estilos visuales (limpios, relajantes, modo foco)
└── app.js          # Lógica del temporizador y control de estados (JS nativo)

-------------------------------------------------------------------------------
3. MAPA DE TAREAS - TROCEADO EN PASOS MINÚSCULOS (Para abordar de uno en uno)
-------------------------------------------------------------------------------

[x] PASO 1: Cimientos del proyecto y Maquetación Base (HTML)
    * Crear la carpeta del proyecto y los 3 archivos vacíos (index, css, js).
    * Vincular el CSS y el JS en el HTML.
    * Estructurar el HTML con dos contenedores principales (`<div id="pantalla-volcado">` y `<div id="pantalla-foco">`).
    * En la pantalla de volcado: Crear el formulario con los inputs para la "Gran Tarea", el "Sub-paso" y el botón "Iniciar Foco".
    * En la pantalla de foco: Crear los elementos de texto para los títulos y el contenedor grande para el cronómetro `00:00`.

[x] PASO 2: Estilo Visual Relajante (CSS)
    * Aplicar un color de fondo pastel o neutro que transmita calma (evitar blancos puros o colores chillones).
    * Centrar el contenido vertical y horizontalmente en la pantalla de forma limpia.
    * Crear una clase CSS `.oculto { display: none; }` para controlar qué pantalla se muestra.
    * Ocultar la pantalla de foco por defecto para empezar a trabajar solo con el diseño del formulario.

[x] PASO 3: Lógica de Intercambio de Pantallas (JavaScript)
    * Capturar los elementos del DOM en `app.js` (inputs, botones, divs de pantalla).
    * Añadir un evento `addEventListener` al botón "Iniciar Foco".
    * Al hacer clic, validar que los campos no estén vacíos.
    * Guardar los textos en variables y aplicar la clase `.oculto` a la pantalla de volcado, eliminándola de la pantalla de foco.
    * Pintar en la pantalla de foco los textos que el usuario escribió.

[x] PASO 4: El Motor del Temporizador (JavaScript)
    * Crear una variable para el tiempo de foco (ej: 20 minutos = 1200 segundos).
    * Crear una función que reste 1 segundo cada vez (usando `setInterval`).
    * Crear una función auxiliar para formatear los segundos en formato bonito `MM:SS` (ej: 1195 segundos -> `19:55`).
    * Pintar el tiempo actualizado en el HTML cada segundo.
    * Detener el `setInterval` cuando el tiempo llegue a `00:00`.

[x] PASO 5: Bloqueo de Descanso Obligatorio (JavaScript/CSS)
    * Cuando el tiempo llegue a `00:00`, cambiar el estado de la app a "Descanso".
    * Modificar el texto en pantalla por un mensaje de descanso: "¡Para! Levántate, estira las piernas y bebe agua."
    * Modificar el fondo de la pantalla (usar un tono suave/verde que indique pausa).
    * Iniciar una nueva cuenta atrás de 5 minutos (300 segundos).
    * Al finalizar el descanso, mostrar un botón de "Siguiente tarea" que devuelva al usuario a la pantalla de volcado limpia.

-------------------------------------------------------------------------------
4. LÍNEA DE META (Criterio de Aceptación para terminar el Proyecto)
-------------------------------------------------------------------------------
El proyecto está 100% COMPLETADO cuando:
1. Abres el archivo `index.html` en el navegador.
2. Escribes una tarea y un sub-paso.
3. Al dar clic al botón, ves solo el sub-paso y el tiempo corriendo.
4. Al llegar a cero, te avisa y te obliga a descansar sin poder saltártelo.
